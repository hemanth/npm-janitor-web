# npm-janitor-web
> Web interface.

![npm-janitor](./npm-janitor.gif)

# quick start

```
$ npm run watch &
$ npm start
```

# commands

* `npm run build` - build for production
* `npm run watch` - automatically recompile during development
* `npm start` - start a static development web server
